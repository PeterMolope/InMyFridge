# 🧊 In My Fridge

A **virtual fridge app** that helps you keep track of the ingredients you have at home and suggests recipes you can cook with them.
Reduce food waste, save money, and discover meals using what’s already in your fridge.

---

## 🚀 Features

* 🧺 **Virtual Fridge Inventory** – Add and manage ingredients in your fridge.
* 🍳 **Recipe Suggestions** – Get recipe ideas based on available ingredients.
* ⏰ **Expiration Tracking** – Track freshness and avoid wasting food.
* 🔍 **Ingredient Search** – Quickly find recipes using specific ingredients.
* 📱 **Simple Interface** – Easy-to-use design for quick meal planning.

---

## 💡 How It Works

1. Add the ingredients you currently have in your fridge.
2. The app analyzes your available items.
3. It suggests recipes you can cook using those ingredients.
4. Choose a recipe and start cooking!

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/smartfridge-recipes.git
```

Navigate to the project folder:

```bash
cd smartfridge-recipes
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
